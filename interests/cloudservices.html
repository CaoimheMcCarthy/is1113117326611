Cloud Services

The term 'cloud services' encapsulates a broad range of web-based technologies that can take the place of traditional methods in terms of storage, development enviroments and software. The primary types of Cloud Services are Software as a Service, Platform as a Service, Infrastructure as a Service and Communication as a Platform. The use of the term 'cloud' to describe these virtualised IT resources came about as cloud was used as a metaphor for the Internet and a standardised cloud-like shape was used to denote a network on telephony schematics.

Cloud services have been used to make this project almost in it's entirety, as I used Cloud 9, a Platform as a Service, to develop my site along with Github, an example of Software as a Service to host my project, and finally Heroku, another Platform as a Service to deploy my website. This is in place of the more traditional method of writing code in a downloadable program such as Notepad and storing the files locally. Our use of several cloud services as part of this project has greatly strengthened my interest in the topic andit has also highlighted their growing importance and use in development.

Why use Cloud Services

Using cloud services provides many benefits in terms of cost, time efficiency, maintenance and security. The method of payment for a cloud service is pay-per-use, meaning the user only has to pay when they actually want to utilise the service. Also, as the user does not actually purchase the infrastrucure, the maintenance of it and the cost associated with this is solely the responsibilty of the vendor.

Future Innovations in Cloud Services

Mobile Backend as a Service (MBaaS): A cloud service is is quickly becoming mainstream in industry, MBaaS provides web and application developers with a means of connecting their apps to backend cloud storage, along with access to user management and social network integration services. Custom software development kits (SDKs) and application programming interfaces (APIs) are used to provide MBaaS across nearly all mainstream platforms, such as iOS, Android and HTML5. There is an ever-growing number of providers of mobile backend as a services, with notable companies doing so including Microsoft Azure and Oracle Cloud.
